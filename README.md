# Cipher service

Serviço responsável pela segurança da aplicação e da conta do usuário.

## Instalação

```bash
$ npm install
```

## Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Rodando testes

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
